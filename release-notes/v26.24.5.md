# v26.24.5

`pbir-cli` (reports plugin): concrete verification path for platforms
without the Desktop bridge:

- On macOS and Linux (and on Windows until the bridge works), the fallback
  is now explicit: `pbir validate --all`, then with the user's permission
  `pbir publish` to a sandbox workspace in Fabric, then inspect the rendered
  report in the browser via the Chrome MCP tools. Replaces the previous
  vaguer "sandbox publishing" wording in workflow step 1 and the
  "When the bridge is unavailable" note
- Versions: reports plugin, pbir-cli skill, and marketplace metadata bumped
  to 26.24.5
