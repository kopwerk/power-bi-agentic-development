# v26.20

- Switched the marketplace to YY.WW versioning (ISO year + week); `0.X.Y` ends at `0.26.1` and continues as `26.20`. Hotfixes append `.N` (e.g. `26.20.1`)
- Elaborated on Copilot CLI support: README install paths, agents renamed from `*.md` to `*.agent.md` (Claude Code still matches them), and an example `useful-stuff/agent-scripts/enable-windows-longpaths.ps1` for the Windows long-path stuff
- Removed the root `.claude-plugin/plugin.json` so the root is treated as a catalog; existing installs should run `/plugin marketplace update power-bi-agentic-development` to clear stale metadata
- Refreshed the `pbir-cli` skill with progressive disclosure; documented `pbir new report --no-title`, `pbir connect --profile`, the active connection context, fail-closed filter validation, visual groups, visual presets, and best practice analyzer syntax
- Refreshed the `fabric-cli` skill: `fab find` is now the primary cross-workspace search; `search_across_workspaces.py` reframed as the governance complement that surfaces fields `fab find` doesn't return; documented the interactive REPL, `fab deploy` (fabric-cicd integration), `fab export --format` for notebooks, lakehouse export/import, full connection CRUD via `fab set`/`fab rm`, and `fab rm --hard`
- Added an ERROR rule to `pbip-validator`: detects when an M expression in `expressions.tmdl` shares a name with a table; Power BI fails the model load with `duplicate member`. Fix is to rename the expression and reference it via `source = #"<Name> Query"`
